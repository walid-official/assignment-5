Important FAQ

What is the Document Object Model (Dom)

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a document (such as HTML or XML) as a tree of objects, allowing programming languages (like JavaScript) to manipulate the content, structure, and style of web pages.

How do you select an element from the DOM?

To select an element from the DOM, you can use methods like `getElementById()`, `getElementsByClassName()`, `querySelector()`, or `querySelectorAll()`. For example, `document.getElementById('myId')` selects an element with the ID "myId", and `document.querySelector('.myClass')` selects the first element with the class "myClass".

what is the event delegation in the context of the dom and why it useful

**Event delegation** is a technique in the DOM where a single event listener is added to a parent element instead of adding multiple listeners to individual child elements. The event bubbles up from the target element (child) to the parent, allowing the parent to handle events triggered by its children. ### Why it's useful: 1. **Efficiency**: It reduces the number of event listeners, improving performance, especially in cases where many elements need similar event handling. 2. **Dynamic Elements**: It works for elements added to the DOM after the event listener is set, ensuring dynamic content is also managed.
